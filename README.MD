# Swift Learn

A personal repository to learn the Swift programming language in a structured and progressive way.

This repository is designed for:
- Learning Swift from basic programming concepts
- Practicing with small examples and mini projects
- Gradually building toward real-world Apple platform applications

## How to Install Swift

Swift is included with Xcode on macOS.

1. Install Xcode from the App Store
2. Open Terminal and verify installation:

   ```bash
   swift --version
```

---

## How to Run the Code

Inside a chapter or project folder:

```bash
swift main.swift
```

For Xcode projects:

* Open the `.xcodeproj` or `.xcworkspace`
* Build and run using Xcode
